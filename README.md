# Proyecto-Personal---Gestion-de-Barberia
Sistema de gestión de local de barbería y peluquería

Este sistema de gestión se presenta como una opción sencilla y fiable para el manejo de un local típico de barbería y peluquería;
incluyendo funciones tales como registro de clientes con su información, empleados contratados, tipos de servicios prestados por
el local, stock actual de productos así como historial de compra de insumos y ventas de productos a los clientes.
El proyecto se realiza completamente en Java, incluyendo además pruebas con JUnit 4 que sustentan las funcionalidades del producto.
Por otra parte, no se descarta la implementación a futuro de nuevas funciones así como también la incorporación de una base de datos
con SQL para mejorar la robustez y escalabilidad de uso del software.
